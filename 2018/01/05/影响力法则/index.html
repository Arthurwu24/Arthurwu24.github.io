所谓的成功，其实就是做影响力

发表于 2018-01-05 | 分类于 个人成长 | | 浏览 次
字数统计: 1,542 字 | 阅读时长 ≈ 5 分钟

1.

对于大众普通人来说，我之前一直认定的成功定义是，短时期内变现能力。

通俗的说就是，用最短的时间挣钱。

这个定义错了吗？没有，你在短时间内挣得钱比别人多，难道不成功吗？肯定是成功的。

但是我现在对成功的定义有了更深层次的定义，我认为现在所谓的成功，就是做影响力。

什么意思？

我可以短期内赚不到钱，但我可以不断的提高我的个人影响力，在它到达一定阙值的时候，再考虑变现，实现比之前零敲细打挣的多10倍的钱。

2.

曾经我在大学的时候，利用空闲时间做过很多活。

学html+css，给人写静态页面网站，当然那个时候的酬薪比现在市面上低了很多；

给人做英语兼职翻译，按小时收费；

给文学网站写稿，每个星期固定几篇，挣稿费。

然后我得到了什么？

我确实通过上面的3种兼职方式得到了很多零花钱，所以我看起来比周边的同学要阔绰许多。会有一些虚荣心成分在里面。

“咦，这小伙子不错，大学就能开始挣钱了。”这是大人的评价，同学不会有这样的言辞评价，但你可以从他们的眼中，看到他们对你的敬佩。

那时候，我确确实实是觉得快乐的。“我凭着自己的本事，大学就可以挣钱了”，我一直都是这么想的，“能有多少人在大学就挣着钱呢？”。

这样的“兼职游击”生活，大约持续了大二一整年。

后来因为想提前修完学分，提前结束大学生涯，早点实习。所以大三的时候就基本没再做这样的事了，而是拼命的修学分。

3.

在社会上打拼了2年，慢慢地也开始看到一些以前接触不到的人了，随着生活阅历的提高、见识的人和事不断增加，看到的成功人士越来越多，我才开始意识到之前那样的做法其实是错的，或者说我并没有很好的利用自己当时的能力。

如果我把会写html+css做网站的能力，通过写稿的方式，去推广自己，打出品牌，也许我挣到的钱会是之前的好几倍。

如果我把英语翻译的能力，转化为辅导他人，或者读国外的深度好文，提高自身能力，也许我挣到的钱会是之前的好几倍。

这个是当时我所意识不到的。

我沉浸于被“金钱”蒙蔽了双眼的喜悦中，只看到了眼前的利益，得到了被长辈和朋友同学钦佩的虚荣心，却从未想着长线发展，从现在来看确实是浪费了当时的大好机会。

现在的社会，简单直观点说，就是分为有影响力阶级和无影响力阶级这两层。

马云、马化腾、李彦宏、刘强东这些互联网大佬，咪蒙、阿何、恶魔奶爸这些自媒体大v，张亮-Leo、刘飞这些知乎大v等，就属于有影响力阶级。

而诸如我等这些普通大众，就属于无影响力阶级，我们说的话可能只能被周边的同事、朋友听见，并不能辐射到更广泛的人群，而有影响力阶级的人则不同，他们发的朋友圈、微博、知乎想法等，会自然而然的被粉丝进行分享传播，让更多的人看到。

4.

那么针对于我们这些普通大众，该如何去做影响力呢？

最直接的方法，就是通过各种方式去让更多的人认识你。

在如今的互联网时代，我们其实有很多渠道，让更多的人知道自己。我个人觉得最好的2种方式是写文章和参加活动。

比如，参加产品经理线下聚会/运营线下聚会等，你会见到一大批同一圈子的人，这个时候普遍的做法是会互相交换微信、手机号联系方式，以后你如果在朋友圈发一些高质量的内容/点评等，就会被更多的人知道，如果能长期保持下去，肯定就会被更多的人知道。

写文章自不用多说，这样具有传播性的互联网时代，你只要写出好的文章，就一定会被更多的人看到。微信公众号、简书、知乎等等这些平台，都可以让有想法的人发挥出自己的能力。

5.

那么，影响力到底对我们会有什么作用？

第一，容易获得合作资源和信任关系

如果你是创业者，你就会发现这是个很大的优势了。有影响力的创业者，在市面上被人所知的人，在拉投资的时候真的就会比很多人要容易得多。对于创业的人来说，大家应该都明白，有时候其实很多企业的倒闭就是差了那么一笔现金，导致公司运转不下去。

第二，遇到问题了，有更大的机会和可能得到解决

比如你有很多粉丝，那么你的粉丝其实是分布在各大领域的。而你一个人不可能样样精通，就像咪蒙，她的小孩要上学，她仍然需要求助于她的粉丝，这就是个不需要花钱找人，很简便有效的方式，通过自己的影响力，获得帮助解决问题的方式

第三，直接变现赚钱

这个就是我之前说的，比起零敲碎打来赚的钱，在你影响力起来后，可能随随便便你开一个教育班、微信群上课，赚的钱会比之前多多少倍。
