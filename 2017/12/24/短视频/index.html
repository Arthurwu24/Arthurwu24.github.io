聊聊直播和短视频(2)

发表于 2017-12-24 | 分类于 产品分析 | | 浏览 次
字数统计: 2,106 字 | 阅读时长 ≈ 7 分钟

前面一篇也说了，直播16年国内市场到达顶峰“网络直播元年”，后期也每况愈下，风口转向了短视频。

相比于直播，短视频有什么优势呢？

前面我们说了直播的劣势，那么相比较下来，直播的劣势就是短视频的优势。

  • 门槛低，成本低，用户上手简单高效

  • 内容质量相比于直播就要高很多

  • 短视频时间短，用户辨别喜好度成本就要低很多

对于平台来说，同样是如此。直播上头部主播的工资年薪千万，有的甚至比平台集团董事长薪酬都要高，而短视频平台的成本就要低很多，不需要签约许多大牌主播拉流量，安心做好内容，培养好平台调性，在现阶段的短视频市场就会自动获取到许多用户。

2017年3月，来疯开始力推短视频

2017年5月，斗鱼上线短视频功能，并设置了标签分类功能

2017年8月，YY推出短视频平台“补刀小视频”，更重视用户的评论互动

2017年9月，花椒宣布今年投入1个亿，签约短视频达人并对优质内容提供额外补贴

各大之前切入直播领域的pc和移动端平台，都开始全面转向短视频。

接下来以市场上前几位知名度较高的产品为例，musical.ly、抖音、快手

musical.ly

musical.ly差点夭折，它的崛起颇有一点“运气成分”。

muse的诞生

2015年，muse的创始人阳陆育决定做musical.ly的想法已经形成，但还没拿到新的融资，他此前靠着个人关系从一个美籍华人那儿找来的150万人民币已所剩无几。团队一共5个人，用了几个月才把这个最新的短视频项目做出来，然后简单地挂在了网上，没做推广——并不是对自己的产品自信爆棚，而是没钱了。

在一个失败的在线教育项目上尝试了近一年后，他们迫切想知道，新项目会收到怎样的市场反馈。

最终出来的mvp（最小化可行产品）只有两个功能：拍15秒的对嘴型音乐短视频，给开发团队写邮件。产品于2014年7月在中美等地同时上线。

这款app在中国并没有火起来，但是在美国却从刚发布的第一天就开始每天有400～500次的下载，这下载量让阳陆育觉得这事是成了。

但因为没有融到资，团队一直在吃老本，最终沦落到2个月没给团队发工资的地步。

还好，muse参加了猎豹傅盛战队的活动，与其他的创业团队互相pk，最终拿到了猎豹的500万的投资，挽救了岌岌可危的muse团队。

有了这笔融资，muse团队开始全力的去做这款产品。

于是，我们看到了，在2015年7月，musical.ly登上了app store美国地区的榜首，这给muse的后期融资带来了极大的影响。

很快的，就有了美国的SIG的B轮融资。

为什么muse成功了？并没有做多大的推广，也没有十分黑科技的功能，何以超越facebook、instagram、snapchat，登顶美国？

  • 虽然没有黑科技，但是融合了音乐、影片、社群三者做出的15s对嘴型创新功能，十分契合国外青少年团体，口口相传，在没宣传的情况下， 引爆了国外市场

  • 普通人可以在平台上成名，获得成功，比如有15岁的女孩Baby Arail有1810万粉丝、小男孩jacob拥有1580万粉丝，这些都是没有名声在外的普通民众。当然后期随着muse的壮大，贝克汉姆、奥尼尔等都在平台上发声过

  • 不单单是一款产品，还形成了一个社区。比如，定期在国外举办聚会，有了自己团体的共同象征museT恤等，「Don’t judge chanllenge」活动，火爆全球

抖音

再说说短视频领域，跟muse差不多类型的抖音。相比于muse在国外市场的大杀特杀，抖音在中国市场也是用户收割机。

2016年9月，今日头条旗下的抖音正式上线，主打新生代的音乐短视频社区。在抖音上，用户可选择一段平台提供的音乐，再通过肢体动作和后期剪辑，创作出一个短视频，时长控制在5-15秒。

2017年3月，抖音下一位id为岳云云的用户模仿岳云鹏的视频被岳云鹏转发，评论“这是我见过最像的”。当时很多抖音用户都以为是岳云鹏本人。

自此，抖音开始火起来了。

在我看来，抖音火起来的原因主要是做好了下面5点。

  • 最开始从musical.ly挖人（毕竟就是从“借鉴”muse起步的，直接从对方团队挖人是最直接有效的），专注PUGC，发布高质量内容，定位自己是媒体音乐短视频

  • 成功冷启动之后，开始购买内容版权，孵化用户，继续产出高质量内容吸引新用户

  • 体量做起来了之后，开始做“任务挑战”，通过内容运营让更多用户产生新的多元化的创作（UGC）

  • 通过今日头条擅长的内容分发，提高平台用户粘性

  • 体量继续扩大之后，由PUGC开始向UGC下沉（微博的二次崛起就是通过PGC向UGC的下沉方式）

快手

提到短视频，不能不提的是中国市场的绝对霸主，快手。

快手的历史

  • 2011.3 快手诞生。当时叫GIF快手，是一款用来制作、分享 GIF 图片的手机应用

  • 2012.11 GIF快手转型。将制作的内容存储为视频，只有在分享到其他平台，如微博时，才转换成 GIF 图片

  • 2013.10 确定短视频社交属性。强化社交能力

  • 2014.11 正式改名为「快手」。同年，快手在 App Store 连续 4 个月居前 50

  • 2015.6 快手总用户突破 1 亿。 完成 C 轮融资，估值 20 亿美金

  • 2016.10 快手总用户达 3 亿

  • 2017.3 快手获得腾讯 3.5 亿美金融资

那么快手成功的原因是什么呢？

  • 你很难看到一二线城市的白领之类的在快手出现，你打开快手第一眼看到的画风可能很难想象它是一个日活3000万的产品，但正是因为它的目标群体是三、四线的，所以它的体量、市场是无比庞大的，产品的入门门槛非常之低

  • 快手的社交关系很弱，但是平台的粘性很强，用户并不会跟着头部的网红走。快手的发现功能使得快手打造了很多自己平台的中小网红，因此快手敢于和主播五五分成，因为主播离不开这个平台。（并不像上一篇所说的映客那样，头部主播走了，就带走了一批用户）

  • 快手的feed流做的很棒，沉浸式的体验，可以让用户刷很久

在这些短视频产品获得成功和关注的同时，我们其实也看到了很多抖音、快手等产品各自存在的一些问题。

比如，抖音的内容形式过于单一，你刷的时候就会发现可能真正受欢迎的永远是那么几首歌，宠物永远是狗和猫等等，内容并没有真正多元化起来。

比如，快手上“low”的气质可能让很多一二线的用户十分不满意，以及快手平台上很多为了出名，各种“自虐”、“自残”的行为等，也被大家广泛的关注。

但纵使这样，我们也得承认，直播热潮过后，短视频已经迎来了春天。在中国这个市场上，抖音和快手，到底谁能跑的更远，还是双双最后都大获成功？

这是个很值得期待的领域。
